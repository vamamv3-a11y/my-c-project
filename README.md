# Учебный проект на языке C

Сборник лабораторных работ по курсу "алгоритмизация программирования" на C.

## Структура проекта

- `include/` — заголовочные файлы (.h)
- `src/` — исходные файлы (.c)
- `main.c` — главный файл с меню

## Лабораторные работы

- Лаба 1 + доп. задание
- Лаба 2 + доп. задание
- Лаба 3 + доп. задание
- Лаба 4 + доп. задание
- Лаба 5 + доп. задание

## Компиляция

gcc -o my_program main.c src/lab1.c src/lab1\(add\).c src/lab2.c src/lab2\(add\).c src/lab3.c src/lab3\(add\).c src/lab4.c src/lab4\(add\).c src/lab5.c src/lab5\(add\).c -Iinclude
